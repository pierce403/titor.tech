# Lindy

If something has survived a long time, it has (usually) discovered how not to die.

In a world of fast-moving AI capabilities, Lindy is a counterweight: **prefer systems that degrade gracefully and can be repaired by normal humans.**

## Lindy heuristics
- Prefer protocols over platforms
- Prefer open formats over proprietary state
- Prefer redundancy over cleverness
- Prefer tools with decades of operational knowledge

## Caveat
Lindy is not “old = good.” It’s “survival pressure created information.” Use it as a filter, not a religion.
