# EigenLayer

Restaking is an attempt to turn Ethereum security into a modular commodity.

## Why it matters
- Security reuse can accelerate new networks/services
- It can also create correlated risk (shared failure modes)

## Questions we track
- Where does restaking create real value vs. leverage loops?
- What are the systemic-risk failure scenarios?
- How do we get transparent, auditable risk disclosure?
