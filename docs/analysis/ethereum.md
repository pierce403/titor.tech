# Ethereum

Ethereum is the most mature programmable settlement layer for building credibly-neutral coordination mechanisms.

## Why it matters
- Smart contracts as *institutional building blocks*
- Account abstraction + smart wallets → better human UX
- L2s as the scaling path (with fragmentation risk)

## Questions we track
- Which primitives increase alignment bandwidth (identity, reputation, attestations)?
- What reduces fraud/MEV-driven extraction?
- What’s the practical path to privacy (ZK) without breaking usability?
